## Dataset

The datasets used in this work include publicly available training data and the Kerry 3D dataset.

- **Training data:** [Google Drive](https://drive.google.com/drive/folders/1FcykAxpqiy2NpLP1icdatrrSQgLRXLP8)
- **Kerry dataset:** [SEG Wiki: Kerry-3D](https://wiki.seg.org/wiki/Kerry-3D?__cf_chl_tk=w8qUKaLiaKwBuH.nRtXMs_AZZv7B5hBcenNyWrZW4k8-1774892030-1.0.1.1-.MufsgKPCxlPAXWl..Tq3XgHikaIJXn23UmXe3Lusnw)

**Note:** The X field data are not publicly available due to confidentiality restrictions.
